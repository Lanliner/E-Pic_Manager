2．预览界面

1.目录

点击此电脑，显示电脑的磁盘目录结构
可以通过目录树访问电脑中的某级目录

2.查看帮助

点击右上角"i"
查看软件使用帮助文档

3.壁纸

点击右上角"□"
可更换软件主题壁纸颜色

4.隐藏

点击右上角"-"
可隐藏界面窗口

5.全屏

点击右上角全屏键，即可显示全屏化界面
再次点击即可回到窗口化界面

6.退出

点击右上角"×"
关闭界面，退出程序
